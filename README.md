# 💊 Pharmaceutical Supply Chain: Intelligent Anomaly Detection

This project utilizes **Unsupervised Machine Learning** to identify supply chain shocks and demand explosions within the pharmaceutical industry, specifically focusing on the structural breaks caused by the COVID-19 pandemic (Q4 2020).

## 🚀 Key Features
- **Anomaly Hunting:** Implementation of **Isolation Forest** to identify "Panic Signatures" in sales data where traditional models saw only noise.
- **Geographical Risk Profiling:** **K-Means Clustering** to segment cities into *Resilient Zones*, *High-Impact Zones*, and *Critical Hotspots*.
- **Interactive Intelligence:** Dynamic visualizations using **Plotly** to transform raw data into actionable visual cockpits.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, Scikit-Learn, Plotly, Seaborn
- **Algorithms:** Isolation Forest (Anomaly Detection), K-Means (Clustering)

## 📊 Business Value
Designed for supply chain directors to pivot from reactive to proactive inventory management. This pipeline provides a data-driven blueprint for prioritizing emergency logistics to high-risk geographical clusters.
